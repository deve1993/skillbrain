# Rebrand Design: SkillBrain ‚Üí Synapse

**Date:** 2026-05-07  
**Author:** Daniel De Vecchi  
**Status:** Approved

---

## Goal

Replace all user-visible occurrences of "SkillBrain" with "Synapse" across the monorepo.  
Package names (`@skillbrain/storage`, `codegraph`) are **out of scope** ‚Äî internal only.

## New Brand Identity

| Before | After |
|--------|-------|
| `SkillBrain` | `Synapse` |
| `Self-hosted AI memory platform` | `The intelligence layer for AI workflows` |
| `SkillBrain Studio` | `Studio` (no brand prefix) |

---

## Scope

### 1. License headers ‚Äî all `.ts` files (~391 occurrences)

```
// BEFORE
SkillBrain ‚Äî Self-hosted AI memory platform

// AFTER
Synapse ‚Äî The intelligence layer for AI workflows
```

### 2. Frontend ‚Äî `packages/codegraph/public/` (~28 occurrences)

- `public/studio/index.html`
  - `<title>SkillBrain Studio</title>` ‚Üí `<title>Studio ‚Äî Synapse</title>`
  - Logo text: `SkillBrain Studio` ‚Üí `Studio`
- `public/studio/studio.js` ‚Äî any "SkillBrain" strings
- `public/studio/connectors.js` ‚Äî any "SkillBrain" strings
- Other dashboard HTML/JS files

### 3. Backend user-visible strings ‚Äî `packages/codegraph/src/`

**`http-server.ts`:**
- Login page: `<h1>SkillBrain</h1>` ‚Üí `<h1>Synapse</h1>`
- Login page subtitle: `Sign in to your account` ‚Äî keep as-is
- Startup banner: `SkillBrain Hub (HTTP mode)` ‚Üí `Synapse (HTTP mode)`
- Fallback page: `SkillBrain Hub` ‚Üí `Synapse`
- Startup console.log output

**`http-server.ts` invite email:**
- `Sei stato aggiunto al team SkillBrain` ‚Üí `Sei stato aggiunto al team Synapse`
- `Dashboard:` URL label ‚Äî keep
- `SkillBrain <noreply@dvesolutions.eu>` ‚Üí `Synapse <noreply@dvesolutions.eu>`

### 4. Connectors

**`connectors/resend.ts`:**
- `from`: `SkillBrain Studio <noreply@...>` ‚Üí `Synapse <noreply@...>`
- Email body HTML: "SkillBrain Studio" ‚Üí "Studio"

**`routes/studio-connectors.ts`:**
- Drive upload description: `'SkillBrain Studio export: ...'` ‚Üí `'Studio export: ...'`

**`connectors/github.ts`:**
- PR body: `Generated by SkillBrain Studio.` ‚Üí `Generated by Studio.`

**`routes/studio-export.ts`:**
- Any "SkillBrain Studio" in export filenames or content

**`studio-runner.ts`:**
- System prompt / any "SkillBrain" references

### 5. README.md (~20 occurrences)

- Title / tagline: `SkillBrain` ‚Üí `Synapse`
- Subtitle: `Self-hosted AI memory platform for teams` ‚Üí `The intelligence layer for AI workflows`
- All body text references
- Badge text where applicable
- Config examples: `SKILLBRAIN_MCP_URL` env var key ‚Äî **keep** (technical/internal, out of scope)

### 6. CODEX.md (root)

- Any "SkillBrain" references

| Database values | No "SkillBrain" stored in DB |

---

## Approach

1. `grep -r "SkillBrain" packages/ README.md CODEX.md` to enumerate all occurrences
2. Edit file by file with targeted replacements (no blind sed ‚Äî some occurrences are in package names that must not change)
3. Build both packages after to verify zero TypeScript errors
4. Manual note: update `docs/images/logo.svg` separately

---

## Success Criteria

- `grep -r "SkillBrain" packages/codegraph/src packages/storage/src packages/codegraph/public README.md` returns zero results
- `pnpm --filter @skillbrain/storage build` passes
- `pnpm --filter codegraph build` passes
- Login page shows "Synapse"
- Studio header shows "Studio"
- README opens with "Synapse"

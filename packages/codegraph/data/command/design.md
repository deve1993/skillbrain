# /design

Solo fase design, senza codice. Coordina i 3 agenti Design Team.

## Uso
```
/design [descrizione progetto]
```

## Workflow
1. @ux-designer → personas, site map, user flows, wireframes
2. @ui-designer → color palette, typography, spacing, component specs
3. @motion-designer → timing tokens, micro-interactions, scroll animations

## Output
Documento unico con: Executive Summary, UX Specs, UI Design System, Motion Guidelines, Next Steps.

## Note
- Non genera codice, solo specifiche design in markdown
- Può precedere `/frontend` per iterazione col cliente
- Output nella lingua dell'input (IT/EN/CZ)
